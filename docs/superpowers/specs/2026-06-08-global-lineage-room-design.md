# 全球女性主义理论年轮重构设计

## 目标

将第三展厅从“女性主义事件时间线”重构为“女性主义理论演化时间线”。用户应能沿六个阶段理解理论问题如何从权利平等，转向结构压迫、劳动与再生产、差异与身份、跨国联盟，以及数字时代对父权制的拒绝实践。

## UX 与视觉约束

保留现有第三展厅的 UX 和视觉语言，不重做 UI：

- 保留横向滚动、上下交错的时间线卡片。
- 保留点击节点后更新右侧详情卡的交互。
- 保留桌面端粘性详情卡和移动端选中后滚动至详情卡。
- 保留现有动画、纸张纹理、档案感、配色、阴影、撕纸边缘和响应式布局。
- 保留中英文切换、导航锚点和路由。

视觉层级从“事件”改为“理论阶段”，但沿用原有组件结构和样式。

## 数据模型

原事件数组替换为六个 `theoryStages`。每个阶段包含：

- `id`
- `period`
- `sortYear`
- `title`
- `coreQuestion`
- `mainTheories`
- `keywords`
- `keyTextsOrEvents`
- `theoreticalBreakthrough`
- `limitations`
- `relatedSchools`
- `sources`
- `accent`

除标识、排序值、颜色和 URL 外，所有用户可见内容均采用现有 `LocalizedText` 中英文结构。

## 六个阶段

1. 1792—1940s：权利平等与公民身份
2. 1949—1970s：私人生活与父权制批判
3. 1970s—1990s：劳动、家务与社会再生产
4. 1980s—2000s：差异、身份与交叉性
5. 1995—2010s：全球治理、跨国联盟与地方转译
6. 2010s—至今：数字女性主义与拒绝父权制实践

6B4T 作为第六阶段的关键实践之一，与韩国数字女性主义、`#MeToo` 和 `Ni Una Menos` 并列，不设置独立猎奇节点。

## 时间线卡片

时间线仍使用原有交错卡片，但主视觉信息调整为：

1. 阶段范围
2. 理论阶段标题
3. 核心问题的简短呈现
4. 代表理论
5. 理论关键词

卡片不再显示地点、地区、事件类型或事件摘要。辅助文本明确节点是“理论阶段”而非“事件”。

## 详情卡

右侧标题从“事件档案卡”改为“理论阶段卡”，内容顺序为：

1. 阶段范围与理论阶段标题
2. 核心问题
3. 代表理论与相关流派
4. 理论关键词
5. 理论突破
6. 理论局限
7. 关键文本 / 事件
8. 资料入口

“关键文本 / 事件”只作为理论论证的支撑材料，排在理论突破与局限之后，不承担详情卡标题或摘要功能。

## 筛选

删除地区筛选。保留理论流派筛选，并由阶段的 `mainTheories` 与 `relatedSchools` 共同决定匹配结果。

筛选项覆盖六阶段实际使用的理论传统，包括自由主义、激进、马克思主义、社会主义、交叉性、后现代、跨国、全球南方与数字女性主义。默认显示全部阶段。

切换筛选后：

- 当前阶段仍在结果中时保持选中。
- 当前阶段不在结果中时选中第一项。
- 无结果时沿用现有空状态视觉，并提供“查看全部”。

## 资料入口

保留详情卡底部的外部资料入口。单一事件来源改为阶段级 `sources` 数组，每个阶段可列出多个可靠入口，例如关键文本、联合国页面、学术机构或运动资料。

链接继续使用新标签页打开，并保留中英文 `资料入口 / SOURCES` 标签。

## 验收标准

- 页面只呈现六个理论阶段，不再呈现零散事件节点。
- 阶段范围显示在原年份位置。
- 右侧标题使用理论阶段标题。
- 卡片优先回答核心问题，并展示代表理论、理论突破和理论局限。
- 关键文本与事件处于辅助层级。
- 地区、地点、事件类型、摘要、形成背景和地区转译字段不再参与渲染。
- 6B4T 位于第六阶段内部。
- 理论流派筛选、双语切换、动画、布局与响应式行为正常。
- 每个阶段保留可访问的资料入口。
- 构建通过，不存在字段名错误、未使用变量或渲染错误。

## 非目标

- 不重做第三展厅 UI。
- 不更改项目路由、导航或其他展厅。
- 不新增地图、地区子页面或事件详情弹窗。
