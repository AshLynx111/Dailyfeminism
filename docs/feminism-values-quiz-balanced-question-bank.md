# 女权主义价值测验：理论张力版 20 题题库

## 设计原则

本测试不测量用户“是否支持平等”，也不设置女权主义者理应同意的正确答案。

每一道题都要求用户在两种女权主义解释框架之间表达相对倾向。用户回答的不是“这个社会问题是否存在”，而是“面对同一个问题，我更接受哪一种解释或行动路径”。

题目继续采用五级量表：

- 1：非常不同意，明显偏向陈述左侧框架
- 2：不同意，较偏向陈述左侧框架
- 3：中立，两种框架之间没有明显倾向
- 4：同意，较偏向陈述右侧框架
- 5：非常同意，明显偏向陈述右侧框架

计分转换为双向轴：

- 1 = -2
- 2 = -1
- 3 = 0
- 4 = +1
- 5 = +2

每道题只测量一个核心理论张力。部分题目采用反向表述，以降低用户习惯性同意造成的偏差。

## 一、制度改革 vs 结构重塑

理论两端：

- 左侧：自由主义女权主义，强调法律、教育、机会、代表性和制度执行。
- 右侧：激进女权主义、社会主义女权主义，强调现有制度中的权力关系本身需要被改变。

### 1. 相比增加女性进入现有机构的机会，改变这些机构本身的权力关系更重要。

- 不同意：偏向制度改革/自由主义女权主义。
- 同意：偏向结构重塑/激进或社会主义女权主义。
- 冲突焦点：女性进入制度，还是改变制度。

### 2. 与其等待社会观念和权力关系发生深层变化，不如优先争取可以立即执行的法律与政策保障。

- 不同意：偏向结构重塑/激进或社会主义女权主义。
- 同意：偏向制度改革/自由主义女权主义。
- 题目方向：反向。
- 冲突焦点：长期结构改变，还是近期制度成果。

### 3. 如果家庭和亲密关系中的性别权力没有改变，仅提高女性的教育和就业机会仍然不够。

- 不同意：偏向制度改革/自由主义女权主义。
- 同意：偏向结构重塑/激进女权主义。
- 冲突焦点：公共机会平等是否足以改变私人领域的不平等。

### 4. 更多女性成为管理者、议员或法官，本身就是推动女性解放最有效的方式之一。

- 不同意：偏向结构重塑/激进或社会主义女权主义。
- 同意：偏向制度改革/自由主义女权主义。
- 题目方向：反向。
- 冲突焦点：代表性政治，还是制度性质。

## 二、性别压迫 vs 阶级压迫

理论两端：

- 左侧：激进女权主义，强调男性支配、身体控制、性暴力和父权制性别权力。
- 右侧：马克思主义女权主义，强调财产、阶级、劳动关系和经济剥削。

### 5. 相比收入和劳动条件，男性对女性身体、安全与亲密关系的控制更能解释女性受压迫的核心。

- 不同意：偏向阶级压迫/马克思主义女权主义。
- 同意：偏向性别压迫/激进女权主义。
- 冲突焦点：身体和性别权力，还是经济和劳动关系。

### 6. 如果经济剥削和阶级差距得到根本改变，许多性别不平等也会随之明显减弱。

- 不同意：偏向性别压迫/激进女权主义。
- 同意：偏向阶级压迫/马克思主义女权主义。
- 题目方向：反向。
- 冲突焦点：阶级改变能否显著解决性别压迫。

### 7. 即使不同阶层的女性处境差异很大，她们仍共同面对一种不能被阶级问题取代的性别权力。

- 不同意：偏向阶级压迫/马克思主义女权主义。
- 同意：偏向性别压迫/激进女权主义。
- 冲突焦点：性别压迫是否具有独立于阶级的解释力。

### 8. 与其把性骚扰和家庭控制主要理解为男性支配，不如把它们放进住房、收入依赖和劳动分工中解释。

- 不同意：偏向性别压迫/激进女权主义。
- 同意：偏向阶级压迫/马克思主义女权主义。
- 题目方向：反向。
- 冲突焦点：性别支配解释，还是物质依赖解释。

## 三、文化价值 vs 劳动分析

理论两端：

- 左侧：文化女权主义，强调照护、共情、合作和女性经验中被贬低的价值。
- 右侧：马克思主义女权主义，强调劳动价值、无偿劳动、工资和资源分配。

### 9. 相比赞美照护与共情的价值，我更关心谁在无偿或低薪地承担照护劳动。

- 不同意：偏向文化价值/文化女权主义。
- 同意：偏向劳动分析/马克思主义女权主义。
- 冲突焦点：重新评价女性化价值，还是分析劳动分配与剥削。

### 10. 护理、育儿和情绪支持的问题，首先是这些能力没有得到文化尊重，而不只是劳动报酬不足。

- 不同意：偏向劳动分析/马克思主义女权主义。
- 同意：偏向文化价值/文化女权主义。
- 题目方向：反向。
- 冲突焦点：文化贬值，还是经济低估。

### 11. 即使照护劳动获得合理报酬，如果竞争和效率仍被视为更高级的价值，女性经验依然会被贬低。

- 不同意：偏向劳动分析/马克思主义女权主义。
- 同意：偏向文化价值/文化女权主义。

## 四、普遍女性经验 vs 差异经验

理论两端：

## 数据结构建议

每道题的数据结构应明确记录理论张力，而不是同时给多个维度随意加分：

```ts
type TensionKey =
  | "reform_structure"
  | "gender_class"
  | "culture_labor"
  | "universal_difference"
  | "stable_fluid"
  | "discourse_material";

type TensionQuestion = {
  id: number;
  zh: string;
  en: string;
  tension: TensionKey;
  direction: 1 | -1;
};
```

- `direction: 1`：同意指向理论张力右侧。
- `direction: -1`：同意指向理论张力左侧。
- 回答先转换为 `-2` 至 `+2`，再乘以 `direction`。
- 同一理论张力内的题目取平均值，得到稳定的双向轴分数。

## 流派映射建议

- 自由主义女权主义：制度改革轴左侧。
- 激进女权主义：制度改革轴右侧 + 性别/阶级轴的性别压迫侧。
- 马克思主义女权主义：性别/阶级轴的阶级侧 + 文化/劳动轴的劳动侧 + 话语/物质轴的物质侧。
- 社会主义女权主义：结构重塑侧，并综合性别压迫与阶级压迫；不要求在性别/阶级轴上走到任何单一极端。
- 文化女权主义：文化价值侧 + 普遍女性经验侧 + 稳定身份侧。
- 后现代女权主义：身份流动侧 + 话语建构侧。
- 交叉性女权主义：差异经验侧，并结合结构重塑倾向。

## 结果解释原则

- 不输出“你答对了多少”或“你的女权主义认同度”。
- 不因为用户总体更常选择同意而增加任何流派分数。
- 结果只解释用户在六组理论张力中的相对位置。
- 主导、次级和影响流派应由六个张力轴综合计算。
- 当多个轴接近中点时，应提示用户的观点具有折中性，而不是强行制造高匹配度。
